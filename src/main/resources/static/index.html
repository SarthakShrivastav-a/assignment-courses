Course Platform API

Backend API for a learning platform — browse courses, enroll, and track progress.

Tech Stack

Java 17 Spring Boot 4.0 PostgreSQL Elasticsearch 8.17 JWT Auth Docker Compose

Key Endpoints

  • POST/api/auth/registerCreate account
  • POST/api/auth/loginGet JWT token
  • GET/api/coursesList all courses
  • GET/api/courses/{id}Course details
  • GET/api/search?q=Full-text search
  • POST/api/courses/{id}/enrollEnroll in course
  • POST/api/subtopics/{id}/completeMark complete
  • GET/api/enrollments/{id}/progressTrack progress
